Sobre mí

Llevo adquiriendo conocimientos de programación desde los 16 años, me he dedicado a ponerlos en práctica en varios proyectos desde entonces, se pueden destacar varias etapas de desarrollo dentro de mi vida. Podría decir que mi inicio fue con LUA, un lenguaje de programación que se utiliza en el juego Garry's Mod, en el cual sobre todo me dediqué a editar plugins ya que mi nivel no era muy avanzado.

Más tarde aprendí Python y me dediqué a crear bots para Discord que están disponibles en mi GitHub, también creé aplicaciones de escritorio con Python muy básicas. Más tarde aprendí HTML y CSS y desarrollé páginas web. A día de hoy estoy aprendiendo JavaScript, he aprendido Java y estoy desarrollándome un poco más en LUA además de querer aprender C# y C++.

Mi experiencia en diseño web es amplia, con HTML5 y CSS3 puedo crear páginas web de forma rápida y eficiente, además de que con JavaScript puedo añadirle funcionalidad a las mismas aunque aún estoy desarrollándome en este lenguaje.

Mi lenguaje de programación favorita es Python, aunque también tengo experiencia en Java y quiero introducirme en C++ y C#. Además de esto, tengo experiencia en desarrollo de aplicaciones de escritorio con Java y Python y desarrollo de bots de Discord.

Me queda desarrollarme aún más en frameworks como NodeJS, Vue y React en los que he realizado tanto cursos como proyectos y los veo bastante útiles para futuro. Además de muchas otras tecnologías que me gustaría dominar en un futuro.
